# Example Vendor Tags

This document demonstrates the application of our **Consistent Tagging Language**.
Each vendor is assigned a standard set of tags following the structure:
`[Vendor Name]`, `[Market (Broad)]`, `[Service (Specific)]`, `[System (Optional)]`

## Multi-Department Retailers
These vendors cover multiple markets.
- **Amazon**
  - `['Amazon', 'Home Goods', 'Electronics', 'Clothing']`
- **Target**
  - `['Target', 'Groceries', 'Home Goods', 'Clothing', 'Electronics']`

## Groceries
- **Whole Foods**
  - `['Whole Foods', 'Groceries']`
- **Safeway**
  - `['Safeway', 'Groceries']`

## Dining & Coffee
